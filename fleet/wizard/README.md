# ✦ Mr. Wizard

> Deep reasoning specialist.

**Mr. Wizard** is the deep reasoning specialist and synthesis engine that lives within the Obsidian Circle council.

## What it does

- Synthesizes deliberations from multiple council members into a coherent final answer.
- Handles the most complex reasoning tasks that require frontier-class model capabilities.
- Acts as the final authority on architectural and logic-heavy decisions.
- Distills technical positions into actionable recommendations.
- Refines and polishes the output for clarity and technical accuracy.

## Key files

| File | Purpose |
|------|---------|
| `wizard.py` | Core synthesis and deep reasoning logic |
| `council_config.yaml` | Configuration for the synthesis model and parameters |

## Talks to

- **[Obsidian Circle](../obsidian_circle/)** — Receives council deliberations for synthesis.
- **[Terminus](../../terminus/)** — Invoked by tools requiring high-confidence reasoning.
- **[Engram](../engram/)** — Stores distilled knowledge for future retrieval.

## Configuration

Synthesis model (e.g., Claude 3.5 Sonnet or Qwen 32B) configured in `council_config.yaml`.
